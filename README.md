# microservice-sample-ItemStore


## پروژه نمونه میکروسرویس بر پایه .NET 8

### توضیحات پروژه

این پروژه نمونه‌ای از یک معماری میکروسرویس است که با استفاده از .NET 8 توسعه یافته است. هدف این پروژه، آموزش و بررسی معماری میکروسرویس‌ها است. این پروژه از ابزارها و تکنولوژی‌های مختلفی برای پیاده‌سازی و مدیریت میکروسرویس‌ها استفاده می‌کند که در زیر به توضیح آن‌ها پرداخته شده است.

### پیش‌نیازها

- **.NET SDK 8**: این پروژه نیاز به .NET SDK 8 دارد. می‌توانید آن را از [صفحه دانلود .NET](https://dotnet.microsoft.com/download/dotnet/8.0) دانلود و نصب کنید.

- **Docker & Docker Compose**: برای اجرای میکروسرویس‌ها از Docker استفاده می‌شود. می‌توانید Docker را از [صفحه دانلود Docker](https://www.docker.com/products/docker-desktop) دانلود و نصب کنید.

- **MongoDB**: این پروژه از MongoDB به عنوان پایگاه داده استفاده می‌کند. می‌توانید اطلاعات بیشتر و لینک‌های دانلود را از [سایت رسمی MongoDB](https://www.mongodb.com/) دریافت کنید.

- **RabbitMQ & MassTransit**: برای پیاده‌سازی پیام‌رسانی میان میکروسرویس‌ها از RabbitMQ و کتابخانه MassTransit استفاده می‌شود. می‌توانید RabbitMQ را از [صفحه دانلود RabbitMQ](https://www.rabbitmq.com/download.html) دانلود و نصب کنید.

- **Identity Server & OpenID Connect**: برای بحث امنیت و مدیریت هویت از Identity Server و پروتکل OpenID Connect استفاده می‌شود. اطلاعات بیشتر را می‌توانید از [سایت IdentityServer](https://identityserver.io/) مطالعه کنید.

- **Seq**: برای تحلیل و مانیتورینگ لاگ‌های سیستم از Seq استفاده می‌شود. اطلاعات بیشتر را از [سایت Seq](https://datalust.co/seq) ببینید.

- **Prometheus & Grafana**: برای مانیتورینگ و مشاهده اطلاعات سیستم از Prometheus و Grafana استفاده می‌شود. می‌توانید Prometheus را از [سایت Prometheus](https://prometheus.io/) و Grafana را از [سایت Grafana](https://grafana.com/) دانلود کنید.

- **React**: برای بخش فرانت‌اند این پروژه از فریمورک React استفاده شده است. اطلاعات بیشتر و لینک‌های دانلود را می‌توانید از [سایت رسمی React](https://reactjs.org/) دریافت کنید.
